# The Clock Drive-In

A website for [The Clock Drive-In](http://clockdrivein.com), 
a Greenville staple.

## Setup

You'll need to specify email addresses in a file called `emails.txt`

Comma separated, one line

Example:

```
123@abc.com, jimmay@southpark.com
```
